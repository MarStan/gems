# Info
Run all commands from container except deployment
## Deployment
```shell
./deploy.sh
```
## Migrations
```shell
php artisan migrate
```
## Seed
```shell
php artisan db:seed
```
## Sync
```shell
php artisan meetings:sync
```

## Send emails
```shell
php artisan emails:send
```
